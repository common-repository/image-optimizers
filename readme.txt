== Installation ==

There are a few options for installing and setting up this plugin.

= Upload Manually =

1. Download and unzip the plugin.
2. Upload the 'image-optimizers' folder into the '/wp-content/plugins/' directory
3. Go to the Plugins admin page and active the plugin.

= Install Via Admin Area =

1. In the admin area go to Plugins > Add New and search "Image Optimizers"
2. Click install and then activate.
